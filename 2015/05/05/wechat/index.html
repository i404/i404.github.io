一次被微信坑的经历

By i404

发表于 2015-05-05

今天突然发现自己的微信公众平台突然挂了，明明什么都没动，竟然就不能用了，甚是郁闷。于是慌忙登录SAE，准备一探究竟。

我的平台是用 Python 的 WeRoBot 框架做的，看错误代码大概的意思是数据格式不符合规范，我猜是微信公众平台的某些API更新了，所以可能有点不兼容了。还好那部分是实现自定义菜单的，本来就没什么用，注释掉以后发现又可以用了。那就等作者更新或者有兴趣了以后自己改一改吧。

后来发现抠门的腾讯竟然免费提供了语言识别的功能，心想反正老夫以前在测试平台上能用，那现在稍微改改就可以了。

后来发现根本不是想象中的那样啊！一直都输出None啊，妈的这并不科学啊。会不会是API更新了呢，和以前不一样了呢？ 于是我就去看文档啊，发现这个接口还是一毛一样啊，丝毫没变啊，妈的那为什么会不行呢？

于是我不能忍啊，不能忍～
想看看输出的信息啊。鱿鱼搭在SAE上的后端程序看输出好像有点难度啊，我就在本地写了个简单的socket的服务端，后端程序做客户端发消息过来。

事实证明，发过来的XML是好的，并不是坏的，也没有加特效。那为什么不行呢呢呢呢？？

接下来就撤下了socket，发现还是不能用啊，那再看看代码有什么问题吧。

啊，找不粗来啊，为什么不行啊，这是为什么啊？。。算了，下次再改吧。

嗯，正当我要放弃时，我特么发现语音识别功能已经可以用了！！ 你特么是不是在逗我啊？ 妈的等于说是你服务器不稳定？？妈的被坑惨了。

看来我还是太年轻了。。。
